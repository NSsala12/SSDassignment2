1. Open the folder where you have installed xampp on your computer.
(Normally it is located in C:\xampp)

2. Go to htdocs (C:\xampp\htdocs)

3. Create a folder called SSDassignment.

4. Copy the files in the downloaded SSDassignment folder to folder you have created.

5. Run xampp.

6. Open the browser and type https://localhost/SSDassignment/index.html and run the application.
